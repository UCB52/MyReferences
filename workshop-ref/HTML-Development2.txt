*****BASİC HTML:
****!DOCTYPE= Tüm HTML dökümanları bu bildiriyle başlamak zorundadır.Bu bir HTML etiketi değildir.Tarayıcıya hangi belge türünün bekleneceği hakkında bir bilgidir.HTML5'te bu bildiriyi yapmak oldukça basittir ve case sensitive değildir!Örnek:
<!DOCTYPE html><html></html>  //true
<!DocType html><html></html>  //true
<!Doctype html><html></html>  //true
<!doctype html><html></html>  //true
****html = HTML belgesinin rootunu temsil eder.Bu etiket, diğer tüm kullanılan HTML etiketlerinin containerıdır(<!DOCTYPE> bildirisi hariç).Web sayfasının dilini belirtmek için de her zaman html etiketi içerisine "lang" attributunu belirtmemiz lazım.Bu şekilde arama motorlarına ve tarayıcılara fazlasıyla yardımcı oluruz.Default özellikleri:
html {
  display: block;
}

html:focus {
  outline: none;
}
****head = Meta verileri(veriler hakkındaki veriler) için bir containerdır ve <html> ile <body> tagları arasında kullanılır.Meta datalar, HTML belgesi hakkındaki verilerdir.Meta taglar sayfalarda görüntülenmezler.Meta datalar genellikle döküman başlığı, charseti, styleı, scriptleri ve diğer meta dataları tanımlar.<head> içerisinde aşağıdaki taglar kullanılabilir:
**title(Her HTML içerisinde zorunludur) 
**style
**base 
**link 
**meta 
**script 
**noscript 
****title = Dökümanın başlığını tanımlar.Başlık salt text olmalıdır ve tarayıcının başlık çubuğunda veya sayfa sekmesinde gösterilir.HTML belgelerinde bu etiket zorunludur!Bir sayfa başlığının içeriği, arama motoru optimizasyonu(SEO) için çok önemlidir.Arama sonuçlarında sayfaları listelerken sırasına karar vermek için arama motoru algoritmaları tarafından kullanılırlar.Browser toolbarında bir başlık tanımlar.Sayfa, sık kullanılarlara eklendiği zamansa sayfa için bir başlık sağlar.Arama motoru sonuçlarında ise sayfa için bir başlık görüntüler.Kaliteli titlelar oluşturmak için birkaç ipucu:
-Daha uzun ve açıklayıcı bir başlık seçin(Herzaman bir veya iki kelimeden daha fazlasını kullanın)
-Arama motorları, başlaığın yalnızca yaklaşık 50-60 karaketerini gösterir.Bu nedenle bu sayıdan daha fazla titlelar yazmayın.
-Başlık olarak yalnızca bir kelime listesi kullanmayın ve anlamlı cümleler yazın.Çünkü arama motorlarındaki konumunu azaltabilir!
***Bir HTML dökümanında birden fazla title kullanamazsınız!
****body = HTML belgesinin gövdesidir.Başlıklar, paragraflar, görseller, hyperlinkler, tablolar, listeler, vs. gibi bir HTML belgesinin tüm içeriğini içerir.Default değeri:
body {
  display: block;
  margin: 8px;
}

body:focus {
  outline: none;
}
***Bir HTML dökümanında birden fazla body kullanamazsınız!
****h1 = Sayfa için başlıklardır.En önemli başlıktır.Default değeri:
h1 {
  display: block;
  font-size: 2em;
  margin-top: 0.67em;
  margin-bottom: 0.67em;
  margin-left: 0;
  margin-right: 0;
  font-weight: bold;
} 
****h2 = Sayfa için başlıklardır.Daha az önemli başlıktır.Default değeri:
h2 {
  display: block;
  font-size: 1.5em;
  margin-top: 0.83em;
  margin-bottom: 0.83em;
  margin-left: 0;
  margin-right: 0;
  font-weight: bold;
}
****h3 = Sayfa için başlıklardır.Daha az önemli başlıktır.Default değeri:
h3 {
  display: block;
  font-size: 1.17em;
  margin-top: 1em;
  margin-bottom: 1em;
  margin-left: 0;
  margin-right: 0;
  font-weight: bold;
}
****h4 = Sayfa için başlıklardır.Daha az önemli başlıktır.Default değeri:
h4 {
  display: block;
  font-size: 1em;
  margin-top: 1.33em;
  margin-bottom: 1.33em;
  margin-left: 0;
  margin-right: 0;

**h5 = Sayfa için başlıklardır.Daha az önemli başlıktır.Default değeri:

***Her sayfa için yalnızca bir tane <h1> kullanın!Böylece tüm sayfanın konusunu temsil edecektir.Daha alt başlıklara indikçe de başlığın düzeyini azaltın: h1,h2,h3,h4,h5,h6 şeklinde.
**p = Paragraf tanımlar.Tarayıcılar default olarak her <p> elementinden öncesine ve sonrasına tek bir boş satır(line) ekler.Default değeri:
p {
  display: block;
  margin-top: 1em;
  margin-bottom: 1em;
  margin-left: 0;
  margin-right: 0;
}
**br = Tek bir satır sonu ekler.Genellikle adres veya şiir yazmak için kullanılır.Boş bir etikettir yani bitiş tagı kullanılmaz!
***<br> etiketini contentler arasına boşluk eklemek için kullanmayın; satır sonu için kullanın!
**hr = HTML sayfasında tematik bir ara çizgi tanımlar.Genellikle contentleri ayırmak için kullanılan horizontal ruledur.Default değeri:
hr {
  display: block;
  margin-top: 0.5em;
  margin-bottom: 0.5em;
  margin-left: auto;
  margin-right: auto;
  border-style: inset;
  border-width: 1px;
}
**<!--...--> = Yorum satırıdır.Bu etiketi içerisinde kullanılanlar sayfada görüntülenmez ve işleme alınmaz.

*****FORMATTİNG HTML TAGS:
**abbr = Bu etiket, kısaltmaları veya tanımlamaları belirtir.Default değeri:
abbr {
    display: inline;
  }
**adress = Bu etiket, bir belgenin veya makalenin yazarı/sahibi için iletişim bilgilerini tanımlar.İletişim bilgileri E-posta adresi, URL, fiziksel adres, telefon numarası, sosyal medya hesapları, vs.. şeklinde olabilir.Genellikle italic olaran renderlanır ve tarayıcılar, her zaman <adress> tagından önce ve sonra bir adet satır sonu ekler.Default değeri:
address {
    display: block;
    font-style: italic;
  }
**b = Texti kalınlaştırır.Fakat bunun yerine her zaman <strong> kullanın çünkü semantik değildir!
**bdi = Bu etiketin anlamı Bi-Directional İsolationdur(Çift yönlü izolasyon).Dışarısındaki diğer textlerden farklı bir yönde biçimlendirilebilecek textin bir bölümünü yalıtır.Kullanıcı tarafından oluşturulan içeriği bilinmeyen bir text yönü ile ekleme yapılırken kullanışlıdır.Mesela arapça veya farsça gibi dillerde sağdan sola yazılma olduğu için çok kullanışlıdır.Örnek:
<ul>
 <li>User <bdi>hrefs</bdi>: 60 points</li>
 <li>User <bdi>jdoe</bdi>: 80 points</li>
 <li>User <bdi>إيان</bdi>: 90 points</li>
</ul>
<-Bunun sonucunda eğer <bdi> etiketi kullanmasaydık çıktısı şöyle olurdu:
User hrefs: 60 points
User jdoe: 80 points
User إيان 90 points
****bdo = Bu etiketin anlamı Bi-Directional Overridedır(Çift yönlü geçersiz kılmak).Geçerli textin yönünü geçersiz kılmak için kullanılır.Alabileceği tek attribute: "dir".
****blockquote = Bu etiket, başka bir kaynaktan alıntı yapılan bir bölümü belirtir.Tarayıcılar genellikle bu içerikleri girintili halde gösterilir fakat CCS ile biçimlendirebiliriz.Kısa alıntılar içinse <q> tagı kullanabiliriz.Alabileceği tek attribute: "cite".Default değeri:
blockquote {
    display: block;
    margin-top: 1em;
    margin-bottom: 1em;
    margin-left: 40px;
    margin-right: 40px;
  }
****cite = Bu etiket, bir çalışmanın(kitap,şiir,şarkı,film,tablo,heykel,vs..) başlığını tanımlar.Eser başlığı olarak şahıs ismi vs kullanmayın!İçeriği genelde italic olarak renderlanır.
-><cite> default değeri:
cite {
    font-style: italic;
  }
****code = Bu etiket, bir bilgisayar kodu parçasını tanımlamak için kullanılır.İçeriğinin yazı tipini değiştirerek monoscape fontta görüntüler.Default değeri:
code {
    font-family: monospace;
  }
****del = Bu etiket, bir belgeden silinen bir texti belirtir.Tarayıcılar genellikle silinen textin üzerinden çizgi çeker.Alabileceği attributeler:
**cite
**datetime
-><del> default değeri:
del {
    text-decoration: line-through;
  }
****dfn = Bu etiket, tanım elementi anlamına gelir ve içeriğinde tanımlanacak olan elementi belirtir.Bu etikete en yakın olan parentı, elementin açıklamasını/tanımını da içermelidir.Genelde içeriği italic şekilde görüntüler.Default değeri:
dfn {
    font-style: italic;
  }
****em = Bu etiket, texti vurgulamak için kullanılır.Genellikle içeriği italic yapar.Ekran okuyucular(screen reader), sesli okuma esnasında bu içerikleri vurgulu okumaktadır.Default değeri:
em {
    font-style: italic;
  } 
****i = Bu etiket, textin bir bölümünü alternatif bir ses veya ruh halinde tanımlar.Genellikle italic olarak görüntülenir.Genellikle bir terimi, başka dilden bir ifadeyi, bir düşünceyi, bir özel ismi, vs.. belirtmek için kullanılır.Fakat tavsiye edilmez çünkü semantik değildir!Default değeri:
i {
    font-style: italic;
  }
****ins = Bu etiket, bir belgeye eklenmiş bir texti tanımlar.Genellikle altılı çizgili olarak görüntülenir.Alabileceği attributeler:
**cite
**datetime
-><ins> default değeri:
ins {
    text-decoration: underline;
  }
****kdb = Bu etiket, klavye inputunu tanımlamak için kullanılır.İçeriği, mososcape fontunda görüntüler.Default değeri:
kbd {
    font-family: monospace;
  }
****mark = Bu etiket, işaretlenmesi veya vurglanması gereken texti tanımlar.Genellikle backgroundu sarı olarak görüntüler.Default değeri:
mark { 
    background-color: yellow;
    color: black;
  }
****meter = Bu etiket, bilinen bir aralıktaki skaler bir ölçümü veya kesirli bir değeri tanımlar.Mesela disk kullanımı vs..Her zaman <label> belirterek kullanın.Fakat ilerlemeyi belirtmek için progress bar olarak kullanılamaz!İlerleme çubukları için <progress> tagını kullanın.Alabileceği attributeler:
**form
**high
**low
**max
**min
**optimum
**value
****pre = Bu etiket, texti formatlamadan olduğu gibi yazar.Yani kod bloğundaki gibi white-spaceleri de tarayıcıda görüntüler.HTML kaynak kodunda yazıldığı gibi görüntülenecektir.Default değeri:
pre {
    display: block;
    font-family: monospace;
    white-space: pre;
    margin: 1em 0;
  } 
****progress = Bir görevin, tamamlanma sürecindeki ilerlemesini belirtir.Her zaman <label> ile kullanın.Javascript ile kullanılırsa daha işlevsel şeyler ortaya çıkabilir.Alabileceği attributeler:
**max
**value
****q = Bu etiket, kısa bir alıntıyı ifade eder.Genelde ifadenin etrafına tırnak işaretleri("") eklenir.Alabileceği tek attribute: "cite".Default değeri:
q {
    display: inline;
  }
  
  q:before {
    content: open-quote;
  }
  
  q:after {
    content: close-quote;
  }
****s = Bu etiket, textin artık doğru veya alakalı olmadığını belirtir.Genellikle textin üzerinde çizgili olarak görüntüler.<del> ile karıştırmayın!Default değeri:
s {
    text-decoration: line-through;
  }
****samp = Bu etiket, bir bilgisayar programından örnek outputu tanımlamak için kullanılır.İçeriği genellikle monoscape fontunda görüntüler.Default değeri:
samp {
    font-family: monospace;
  }
****small = Bu etiket, texti daha küçük halde görüntüler(telif hakkı veya yan yorumlar gibi).Default değeri:
small {
    font-size: smaller;
  }
****strong = Bu etiket, texti daha kalın yapar.Default değeri:
strong {
    font-weight: bold;
  } 
****sub = Bu etiket, alt simge textini tanımlar.Normal satırın yarım karakter altında görünür ve daha küçük bir yazı fontuyla işlenir.Default değeri:
sub {
    vertical-align: sub;
    font-size: smaller;
  }
****sup = Bu etiket, üst simge textini tanımlar.Normal satırın yarım karakter üstünde görünür ve daha küçük bir yazı fontuyla işlenir.Default değeri:
sub {
    vertical-align: sup;
    font-size: smaller;
  }
****template = Bu etiket, sayfa yüklendiğinde kullanıcıdan gizlenen bazı HTML içeriğini tutmak için bir container olarak kullanılır.İçerisindeki içerik daha sonra bir JS ile oluşturulabilir.Bu etiketi, tekrar tekrar kullanmak istediğiniz bazı HTML kodlarınız varsa, ancak siz isteyinceye kadar kullanabilirsiniz.Bunu eğer <template> tagı olmadan yapmak için ise JS kullanmamız gerekir.
****time = Bu etiket, belirli bir saati(veya datetime) tanımlar.Bu tagın "datetime" attributu, time kısmını makine tarafından okunabilir bir biçime dönüştürmek için kullanılır ve böylece browserlar, kullanıcının takvimi aracılığıyla tarih hatırlatıcıları eklemeyi teklif edebilir ve arama motorları daha akıllı arama sonuçları üretebilir.Alabileceği tek attribute: "datetime".
****u = Bu etiket, yanlış yazılmış sözcükler veya Çince metinlerdeki gibi özel adlar gibi, eklemlenmemiş ve normal metinden farklı biçimlendirilmiş textleri belirtir.Genellikle altı çizgili olarak görüntüler.Default değeri:
u {
    text-decoration: underline;
  }
*****Hyperlink belirtilebilecek durumlarda <u> tagı kullanmayın çünkü kullanıcı, birbiriyle karıştırabilir!
****var = Bu etiket, programlamada veya matematiksel bir ifadede değişken tanımlar.Genellikle italic olarak görüntüler.Default değeri:
var {
    font-style: italic;
  }
****wbr = Bu etiket, bir textin neresine satır sonu eklemenin uygun olacağını belirtir.Eğer kelime çok uzun olduğunda tarayıcı onu yanlış yerden kesebilir.Fırsatlar yaratmak için bu tagı kullanabiliriz.
*****<bwr> ile <br> arasındaki fark şudur: <br>, belirtildiği noktada mutlak satır sonu uygular; <wbr> ise bunun tercihini browsera bırakır; sadece zorunlu durumlarda satır sonu uygular.

**media = Values:
-media_query
**type = Values:
-media_type (text/css)
-><style> default değeri:

  display: none;
}
****div = Bu etiket, HTML belgesindeki bir parçayı veya sectionı tanımlar.HTML elementleri için container olarak kullanılır.CSS ile biçimlendirilebilir.JS ile manipüle edilebilir.İçerisine her türlü içerik eklenebilir.Default olarak browserlar, herzaman <div> tagından öncesine ve sonrasına bir adet satır sonu ekler.Default değeri:
style {
  display: block;
}
****span = Bu etiket, bir textin bir bölümünü veya bir belgenin bir bölümünü işaretlemek için kullanılan bir inline containerdır.CSS ile biçimlendirilebilir ve JS ile manipüle edilebilir.
****header = Bu etiket, kullanıcıya tanıtım içeriği için bir container veya bir navigasyon linkini temsil eder.Genelde içerdiği şeyler bir veya daha fazla başlık elementi(h1,h2...h6), logo, simge veya yazarlık bilgileridir.HTML dökümanı içerisinde birden fazla kez kullanılabilir.
****footer = Bu etiket, bir belge veya bölüm için alt bilgi sağlar.Genelde içerdiği şeyler yazarlık bilgileri, telif hakkı bilgileri, iletişim bilgileri, site haritası, üst bilgiye geri dönüş veya ilgili dökümantasyonlardır.Bir döküman içerisinde birden fazla kez kullanabiliriz.İçerisinde <adress> tagını kullanılması doğrudur.Default değeri:
footer {
  display: block;
}
****main = Bu etiket, bir belgenin ana içeriğini belirtir.İçeriği bu belgeye özgü olmalıdır; başka tekrarı bulunmamalıdır.Sidebarlar, navigasyon linkleri, site logoları veya arama formları gibi belgeler arasında tekrarlanan herhangi bir içerik içermemelidir.Bir belge içerisinde birden fazla <main> tagı bulunamaz!
****section = Bu etiket, bir belgedeki sectionları(bölümleri) tanımlar.Default değeri:

  display: block;

****article = Bu etiket, belgeden bağımsız kalabilen ve kendi kendine yetebilen içeriği tanımlar.Bir article kendi başına anlamlı olabilmeli ve web sitesinin geri kalanından bağımsız olarak dağıtılabilir olmalıdır.Genellikle kullanım alanları forum postu, blog postu veya haber içerikleridir.Browser tarafından özel bişey olarak renderlanmaz fakat CSS ile biçimlendirilebilir.Default değeri:
article {
  display: block;
}
****aside = Bu etiket, yerleştirildiği containerın dışında bazı içerikleri tanımlar.Fakat etrafındaki bilgilerle dolaylı da olsa bağlantılı olmalıdır.Genellikle belgelerde sidebar olarak kullanılır.Browser tarafından özel bişey olarak renderlanmaz fakat CSS ile biçimlendirilebilir.Default değeri:
article {
  display: block;
}
****details = Bu etiket, kullanıcının istek üzerine açıp kapatabileceği ek ayrıntıları belirtir.Default olarak widget kapalıdır.Açıldığında genişler ve içeriği görüntüler.Ayrıntılar için başlık belirtmek içinse içerisinde <summary> tagı kullanın.Alabileceği tek attribute : "open".Default değeri:
details {
  display: block;
}
****dialog = Bu etiket, bir dialog boxı veya subwindow tanımlar.Bir web sayfasında açılır dialog boxları ve modalları(subwindow) oluşturmayı kolaylaştırır.Alabileceği tek attribute : "open".
****summary = Bu etiket, <details> tagı için görünür bir başlık sağlar.Ayrıntıları görüntülemek veya gizlemek için bu başlığa tıklanabilir.Fakat <details> tagının first chıldı olmak zorundadır.Default değeri:
summary {
  display: block;
}
****data = Bu etiket, belirli bir içeriğin makine tarafında okunabilir bir translateini eklemek için kullanılır.Hem veri işlemcileri için makine tarafından okunabilir bir değer hem de bir tarayıcıda renderlamak için insan tarafından okunabilir bir değer sağlar.Eğer içerik saat veya tarih ile ilgiliyse bu tag yerine <time> tagını kullanın.Alabileceği tek attribute : "value:machine_readable_format".

**href = Values:
-URL

*****GLOBAL ATTRİBUTES:
****accesskey = Bu attribute, bir elementi focuslamak/aktif etmek içn bir shortkey belirtir.Aldığı value tek bir karakter veya tek rakam olmalıdır.Shortkeye erişim farklı browserlarda farklı şekillerde olmaktadır.
****class = Bu attribute, bir element için bir veya daha fazla class adını belirtir.Çoğunlukla CSS ile bir classa başvurmak için kullanılır.JS ile de manipüle edilebilir.
****contenteditable = Bu attribute, bir elementin içeriğinin editlenebilir olup olmadığını belirtir.Alabileceği valueler:
-true
-false
****data-* = Bu attribute, sayfaya veya uygulamaya custom private dataları depolamak için kullanılır.Tüm HTML elementlerinde kullanılabilir.Depolanan custom datalar daha sonra(herhangi bir ajax çağrısı veya server-side veritabanı sorgusu olmadan) daha ilgi çekici bir kullanıcı deneyimi oluşturmak için sayfanın JS'İnde kullanılabilir."data-" prefixine sahip custom attributeler, user agentlar tarafından tamamen yok sayılırlar.Bu attribute iki bölümden oluşur: bu attributeun, adı büyük harf içermemeli ve "data-" prefixinden en az bir karakter uzunluğunda olmalıdır; attribute valuesi herhangi bir string olabilir.
****dir = Bu attribute, elementin içeriğinin text yönünü belirtir.Alabileceği valueler:
-ltr
-rtl
-auto
****draggable = Bu attribute, bir elementin sürüklenebilir olup olmadığını belirtir.Genellikle Drag and Drop işlemlerinde kullanılır.Alabileceği valueler:
-true
-false
-auto
***Linkler ve imageler default olarak "draggable"dır.
****hidden = Bu attribute, elementin görüntülenmemesi gerektiğini belirtir.Booleandır.Genellikle JS ile manipülasyon yapılırken kullanılır.
****id = Bu attribute, bir HTML elementi için unique bir kimlik belirtir(HTML belgesinde eşsiz olmalıdır).Genellikle JS ile manipülasyon için kullanılır.
****lang = Bu attribute, elementin içeriğinin dilini belirtir.Mesela en(İngilizce), es(İspanyolca), fr(Fransızca), vs..Alabileceği tek attribute : "language_code".
****spellcheck = Bu attribute, elementin yazım ve grammerinin denetlenip denetlenmeyeceğini belirtir.Yazım denetimi yapılabilecekler: input elementlerindeki text valueleri(password hariç), <textarea> elementindeki textler ve editable elementlerdeki textlerdir.Alabileceği valueler:
-true
-false
****style = Bu attribute, bir inline CSS belirtir.Eğer burada belirtilmiş propertilerin aynısı, başka yerde kullanılmış olsa bile buradaki propertiler geçerli olur.Alabileceği tek value : "style_definitions (CSS_Properties)".
****tabindex = Bu attribute, elementin tab sırasını belirtir.
****title = Bu attribute, bir element hakkında ek bilgi belirtir.Genellikle mouse ile elementin üzerine gelindiğinde bir tooltip text olarak belirtilir.Alabileceği tek value: "text".
****translate = Bu attribute, elementin içeriğinin translate edilip edilmeyeceğini belirtir.Alabileceği valueler:
-yes
-no

*****WİNDOW EVENT ATTRİBUTES: Window objecti için gerçekleşen eventlerdir(<body> tagına uygulanır).
****onafterprint = Bu attribute, bir sayfa print edilmeye başladığında veya print dialog boxı kapatıldığında etkinleşir.Genellikle "onbeforeprint" attributeu ile beraber kullanılırlar.Alabileceği tek value: "script".
****onbeforeprint = Bu attribute, bir sayfa print edilmek üzereyken(print dialog boxı henüz görünmeden önce) etkinleşir.Genellikle "onbeforeprint" attributeu ile beraber kullanılırlar.Alabileceği tek value: "script".
****onbeforeunload = Bu attribute, belge boşaltılmak(çıkılmak veya refreshlenmek) üzereyken etkinleşir.Kullanıcıya mevcut sayfadan ayrılmak mı yoksa kalmak mı istediğini bildirmek için bir confirm boxı şeklinde bir mesaj görüntülememizi sağlar.Confirm boxta gösterilen default mesaj: "Are you sure you want to leave this page?" şeklindedir ve bu mesaj değiştirilemez.Fakat bu defult mesajla beraber ekstra custom mesaj da yazabiliriz.Alabileceği tek value : "script".
****onerror = Bu attribute, external bir dosya(mesela belge veya resim) yüklenirken bir hata olduğunda etkinleşir.Alabileceği tek value : "script".
****onhashchange = Bu attribute, geçerli URL'nin bağlantı parçasında("#" sembolüyle başlarlar) değişiklikler olduğunda etkinleşir.Etkinleştirmek için birkaç metod vardır: Location objectinin location.hash veya location.href propertilerini ayarlayarak bağlantı parçasını değiştirerek; geçerli sayfaya ileri veya geri butonlarını kullanarak farklı bir bookmark ile giderek; bookmark bağlantısına giden linke tıklayarak.Alabileceği tek value: "script".
****onload = Bu attribute, bir object yüklendiğinde etkinleşir.Bir web sayfası tüm içeriği(görseller,scriptler,CSS,vs..) tamamen yükledikten sonra bir scripti execute etmek için <body> elementi içerisinde kullanılır.Fakat bazı diğer elementlerin içerisinde de kullanılabilir.Ziyaretçinin browser türünü ve sürümünü kontrol etmek ve bilgilere göre web sayfasının uygun sürümünü yüklemek için kullanılabilir.Çerezlerle başa çıkmak için de kullanılabilir.Alabileceği tek value: "script".
****onmessage = Bu attribute, mesaj tetiklendiği zaman etkinleşir.Alabileceği tek attribute : "script".
****onoffline = Bu attribute, browser offline çalışmaya başladığında etkinleşir."ononline" attributeunun tam tersidir.Alabileceği tek value : "script".
****ononline = Bu attribute, browser online çalışmaya başladığında etkinleşir."onoffline" attributeunun tam tersidir.Alabileceği tek value : "script".
****onpagehide = Bu attribute, kullanıcı bir sayfadan ayrıldığında etkinleşir.Alabileceği tek value : "script".
****onpageshow = Bu attribute, kullanıcı bir web sayfasına gittiğinde etkinleşir.Sayfa ilk yüklendiğinde "onload" eventinden hemen sonra gerçekleşir.Alabileceği tek value: "script".
***"onload" ve "onpageshow" arasındaki farklar: "onpageshow", "onload" eventinin hemen ardından gerçekleşir ve sayfa her yüklendiğinde etkinleşir fakat "onload", sayfanın önbellekten yüklendiği durumlarda etkinleşmez!
****onpopstate = Bu attribute, Window'un geçmişi değiştirdiğinde(temizlediğinde) etkinleşir.Alabileceği tek value: "script".
****onresize = Bu attribute, browser penceresi yeniden boyutlandırıldığında(küçültme/büyültme) etkinleşir.Alabileceği tek value: "script".
****onstorage = Bu attribute, Web depolama(storage) alanı güncellendiğinde etkinleşir.Alabileceği tek value: "script".
****onunload = Bu attribute, bir sayfadan ayrıldığında(linke tıklayarak, form göndererek, vs..) veya browser penceresi kapatıldığında etkinleşir.

*****FORM EVENT ATTRİBUTES: HTML formu içerisindeki eylemler tarafından tetiklenen eventlerdir(neredeyse tüm HTML elementleri içindir fakat genellikle form elementlerinde kullanılır).
****onblur = Bu attribute, elementin focusunu kaybettiği an etkinleşir.Genellikle form validation koduyla beraber kullanılır(mesela kullanıcı, form alanından ayrıldığında)."onblur" ile "onfocus" birbirinin tersidir.

****oninvalid = Bu attribute, submit edilebilir bir <input> elementi invalid(geçersiz) olduğunda etkinleşir.Alabileceği tek value: "script".
****onreset = Bu attribute, form resetlendiği zaman etkinleşir.Alabileceği tek value: "script".
****onsearch = Bu attribute, kullanınıcının <input type="search"> elementinde "ENTER" tuşuna bastığında veya arama butonuna tıkladığında etkinleşir.Alabileceği tek value: "script".
****onselect = Bu attribute, bir elementin içerisinde bazı textler seçildikten sonra etkinleşir.Alabileceği tek value: "script".
