*****ARRAYS:
*****concat() = Bu metod, iki veya daha fazla arrayi birleştirir.Mevcut arrayleri değiştirmez ancak birleştirilmiş arraylarin değerlerini içeren yeni bir array döndürür.Herzaman bir ana array bulundurur ve sonrasında diğer arrayleri ekler.Syntaxı:
-array1.concat(array2, array3, ..., arrayX)
*****constructor = Array prototipini oluşturan fonksiyonu döndürür.Nesneler, şu constructor valuelerini döndürür:
-Arraylar için dönen constructor valuesi: function Array() { [native code] }
-Objectler için dönen constructor valuesi: function Object() { [native code] }.
->Suyntaxı:
-array.constructor
*****copyWithin() = Bu metod, mevcut valuelerin üzerine yazarak array elementlerini, bir arraydeki başka bir konuma kopyalar.Arraya asla element eklemez ve lengthini değiştirmez; sadece elementin yerini değiştirir.Orjinal arrayin üzerine yazar, değiştirir ve valueyi döndürür.Syntaxı:
-array.copyWithin(target, start, end)
*****entries() = Bu metod, key/value çiftleriyle yeni bir Array Iterator objecti döndürür.Orjinal arraydeki her item için yeni yineleme objecti, "key" olarak indexi ve "value" olarak item valuesini içeren bir array içerecektir.Arrayde bulunan her element için fonksiyonu birer kez execute eder.Orjinal arrayi değiştirmez.
Mesela [0, "Banana"][1, "Orange"][2, "Apple"][3, "Grapes"] şeklinde arraylar döndürür.Syntaxı:
-array.entries()
*****every() = Bu metod, bir arraydeki tüm elementler, fonksiyon olarak sağlanan bu testi geçerse, "true" valuesini döndürür.Fakat bu teste uymayan bir array elementine denk gelirse geri kalan elementleri kontrol etmez ve "false" döndüdür.Arrayde bulunan her element için fonksiyonu birer kez execute eder.Booleandır.Boş arrayler için execute edilmez.Orjinal arrayi değiştirmez.Syntaxı:
-array.every(function(currentValue, index, arr), thisValue)
*****fill() = Bu metod, bir arraydeki belirtilen elementleri static bir valueyle doldurur.Doldurmanın nerede başlayıp nerede biteceğini belirleyebiliriz; eğer belirtilmezse de tüm array elementleri doldurulur.Orjinal arrayi değiştirir, üzerine yazar ve döndürür.Syntaxı:
-array.fill(value, start, end)
*****filter() = Bu metod, fonksiyon olarak sağlanan bir testi geçen tüm array elementleriyle dolu yeni bir array oluşturur.Arrayde bulunan her element için fonksiyonu birer kez execute eder.Boş arrayler için execute edilmez.Orjinal arrayi değiştirmez.Syntaxı:
-array.filter(function(currentValue, index, arr), thisValue)
*****find() = Bu metod, fonksiyon olarak sağlanan bir testi geçen ilk array elementinin valuesini döndürür.Arrayde bulunan her element için fonksiyonu birer kez execute eder.Eğer fonksiyonun "true" valuesi döndürdüğü bir array elementi bulursa, o array elementinin valuesini döndürür ve geriye kalan valueleri kontrol etmez.Eğer hiç bulamazsa da "undefined" döndürür.Boş arrayleri execute etmez.Orjinal arrayi değiştirmez.Syntaxı:
-array.find(function(currentValue, index, arr),thisValue)
*****findIndex() = Bu metod, fonksiyon olarak sağlanan bir testi geçen ilk array elementinin indexini döndürür.Arrayde bulunan her element için fonksiyonu birer kez execute eder.Eğer fonksiyonun "true" valuesi döndürdüğü bir array elementi bulursa, o array elementinin indexini döndürür ve geriye kalan valueleri kontrol etmez.Eğer hiç bulamazsa da "-1" döndürür.Boş arrayleri execute etmez.Orjinal arrayi değiştirmez.Syntaxı:
-array.find(function(currentValue, index, arr),thisValue)
*****forEach() = Bu metod, bir arraydeki her element için sırayla birer kez fonksiyon çağırır ve uygular.Boş arrayler için execute edilemez.Syntaxı:
-array.forEach(function(currentValue, index, arr), thisValue)
*****from() = Bu metod, "length" propertisine sahip herhangi bir objecti veya iterable herhangi bir objecti kopyalayarak, yeni bir Array objecti döndürür.Syntaxı:
-Array.from(object, mapFunction, thisValue)
*****includes() = Bu metod, eğer array, belirtilen elementi içeriyorsa "true" döndürür; aksi takdirse "false" döndürür.Case sensitivedir.Syntaxı:
-array.includes(element, start)
*****indexOf() = Bu metod, array içerisinde belirtilen itemi arar ve indexini döndürür.Aramaya başlanılan konum belirlenebilir fakat belirtilmezse, en baştan aramaya başlar ve arrayin sonuna kadar devam eder.Eğer item bulunamazsa "-1" döndürür.Eğer item birden fazla kere mevcutsa, ilk bulduğunun indexini döndürür.Syntaxı:
-array.indexOf(item, start)
*****isArray() = Bu metod, eğer belirtilen object bir array ise "true"; değilse "false" döndürür.Syntax:
-Array.isArray(obj)
*****join() = Bu metod, arrayi bir string olarak döndürür.Elementler belirlenen ayraçlarla ayrılır.Default ayraç 
virgüldür(,).Orjinal arrayi değiştirmez.Syntax:
-array.join(separator)
*****keys() = Bu metod, bir arrayin keyleriyle birlikte yeni bir Array Iterator objecti döndürür.Orjinal arrayi değiştirmez.Mesela array objectinin keyleri indexleridir; object objectlerinin keyleri ise propertyleridir.Syntaxı:
-array.keys()
*****length() = Bu metod, bir arraydeki elementlerin sayısını belirler ve döndürür.Syntaxı:
-array.length()
*****lastIndexOf() = Bu metod, array içerisinde belirtilen itemi arar ve indexini döndürür.Aramaya başlanılan konum belirlenebilir fakat belirtilmezse, en sonundan aramaya başlayarak arrayin başına kadar devam eder fakat indexi başlangıç konumuna göre döndürür.Eğer item bulunamazsa "-1" döndürür.Eğer item birden fazla kere mevcutsa, son bulduğunun indexini döndürür.Syntaxı: 
-array.lastIndexOf(item, start)
*****map() = Bu metod, her array elementi için çağrılan fonksiyonun sonuçlarıyla yeni bir array oluşturur ve döndürür.Sağlanan bu fonksiyon, arrayin her elementi için birer kez çağrılır.Boş arrayler için execute edilmez.Orjinal arrayi değiştirmez.Syntax:
-array.map(function(currentValue, index, arr), thisValue)
*****pop() = Bu metod, arrayin son elementini çıkartır ve çıkartılan elementi döndürür.Arrayin lengthini değiştirir.Arrayin içerisinde element kalmayana kadar art arda kullanılabilir.Syntaxı:
-array.pop()
*****prototype = Tüm javascript objectleri için kullanılabilen genel bir constructordur.Global olan Array() objectine başvurur.Arraylere yeni propertiler ve metodlar eklememizi sağlar.Yeni bir properti oluştururken, "TÜM" arrayler bu propertiyi ve valuesini alır.Yeni bir metod oluştururken, "TÜM" arrayler bu metodu alır.Syntaxı:
-Array.prototype.name = value
*****push() = Bu metod, arrayin sonuna yeni elementler ekler.Arrayin lengthini değiştirir ve son halindeki lengthi döndürür.Syntaxı:
-array.push(item1, item2, ..., itemX)
*****reduce() = Bu metod, arrayin her değeri için soldan sağa doğru bir reducer[azaltıcı] fonksiyonu execute edilir.Fonksiyonun dönüş değerleri string veya number olarak bir hesaplayıcıda saklanır[result/total].Sonucunda ise fonksiyonun hesaplanan sonucu olan tek bir value döndürür.Boş arrayler için execute edilmez.Orjinal arrayi değiştirmez.Syntaxı:
-array.reduce(function(total, currentValue, currentIndex, arr), initialValue)
*****reduceRight() = Bu metod, arrayin her değeri için sağdan sola doğru bir reducer[azaltıcı] fonksiyonu execute edilir.Fonksiyonun dönüş değerleri string veya number olarak bir hesaplayıcıda saklanır[result/total].Sonucunda ise fonksiyonun hesaplanan sonucu olan tek bir value döndürür.Boş arrayler için execute edilmez.Orjinal arrayi değiştirmez.Syntaxı:
-array.reduceRight(function(total, currentValue, currentIndex, arr), initialValue)
*****reverse() = Bu metod, arraydeki elementlerin index sıralarını tersine çevirir.Orjinal arrayin üzerine yazar.Syntaxı:
-array.reverse()
*****shift() = Bu metod, arrayin ilk elementini kaldırır ve kaldırılan elementi döndürür.Orjinal arrayi değiştirir.İçerisinde element kalmayana kadar art arda kullanılabilir.Syntaxı:
-array.shift()
*****slice() = Bu metod, arrayin içerisindeki belirlenmiş elementleri yeni bir array olarak döndürür.İndex olarak belirlenen başlangıç konumu includeddır fakat bitiş konumu değildir.Orjinal arrayi değiştirmez.Syntaxı:
-array.slice(star,end)
***Slice ile substring arasındaki en büyük fark şudur: substringin ilk parametresi ikincisinden daha büyük olabilir; js bunu yine başlangıç-bitiş olarak hesaplar fakat slice metodunun ilk parametresi ikincisinden büyük olamaz!
*****some() = Bu metod, arraydeki elementlerden herhangi birisinin fonksiyon olarak sağlanan bu testi geçip geçmediğini kontrol eder.Bu fonksiyon, arraydeki her element için birer kez execute edilir.Eğer fonksiyonun "true" valuesi döndürdüğü bir array elementi bulunursa, kalan değerler kontrol edilmez ve "true" sonucunu alırız.Eğer bulamazsa "false" döndürür.Boş arrayler için execute edilmez.Orjinal arrayi değiştirmez.
*****sort() = Bu metod, arrayin elementlerini UNICODE olarak sıralar.Bu sıralamalar alfabetik veya sayısal olarak artan veya azalan şeklinde olabilir.Fakat default olarak elementleri alfabetik ve artan düzende sıralayacaktır.Fakat stringleri sıralarken dikkatli olun.Mesela "20" stringi, "150" string değerinden büyüktür.Mesela Türkçe karakterleri sıralarken hatalar yapacaktır.Compare fonksiyonları sayesinde bu yanlışlıklar önlenebilir.Orjinal arrayin üzerine yazar.Syntax:
-array.sort(compareFunction)
*****splice() = Bu metod, array elementlerini ekler veya kaldırır ve kaldırılan elementleri döndürür.Orjinal arrayin üzerine yazar ve lengthini değiştirir.Syntaxı:
-array.splice(index, howmany, item1, ....., itemX)
*****toString() = Bu metod, tüm array valuelerini virgülle ayrılmış olarak ve arrayi da bir string olarak döndürür.Orjinal arrayi değiştirmez.Syntaxı:
-array.toString()
*****unshift() = Bu metod, arrayin başına yeni elementler ekler ve yeni lengthini döndürür.Orjinal arrayin üzerine yazar.Syntaxı:
-array.unshift(item1, item2, ..., itemX)
*****valueOf() = Bu metod, arrayin kendisini döndürür.Zaten array nesnesinin default olan bir metodudur.Orjinal arrayi değiştirmez.Syntaxı:
-array.valueOf()

*****BOOLEAN:

function Boolean() { [native code] }.Syntaxı:
-boolean.constructor
*****prototype = Tüm javascript objectleri için kullanılabilen genel bir constructordur.Global olan Boolean() objectine başvurur.Booleanlara yeni propertiler ve metodlar eklememizi sağlar.Yeni bir properti oluştururken, "TÜM" booleanlar bu propertiyi ve valuesini alır.Yeni bir metod oluştururken, "TÜM" booleanlar bu metodu alır.Syntaxı:

*****CLASSES:
*****constructor() = Bu metod, bir "class" içerisinde oluşturulan objectleri oluşturmak ve initialize etmek için özel bir metoddur.Bir class initialize edildiğinde otomatik olarak çağrılır ve tam ismi "constructor" olmalıdır.Aslında bir constructor metodunuz yoksa Javascript, görünmez ve boş bir constructor metodu ekler.Bir classın birden fazla constructor metodu olamaz.Syntaxı:
-constructor(parameters)
*****extends = Bu keyword, bir parent classın altında bir child classı oluşturur.Bu child metod, tüm propertileri, valueleri ve metodları başka bir parent sınıftan kalıtımla(inherit) devralır.Ve ardından parent classtan devraldığı tüm properti ve metodlara erişebilmek için de "super" metodunu kullanırız ve parenta başvururuz.Bu metod, mevcut classın metod ve propertilerini, yeni oluşturulan classlarda kullanabilmek için oldukça faydalıdır.Syntaxı:
-class childClass extends parentClass
*****static = Bu keyword, classlar için static metodları tanımlar.Static metodlar, classın bir nesnesini oluşturmaya gerek duymadan, doğrudan class içerisinden çağrılır.Syntaxı:
-static methodName()
*****super = Bu keyword, parent classa başvurur.Parent classının constructor metodunu çağırmak, propertilerine ve metodlarına erişmek için kullanılır.Syntaxı:
-super(arguments);  //calls the parent constructor(only inside the constructor)
-super.parentMethod(arguments);  //calls a parent method

*****ERROR:
*****name = Bu property, bir errorun adını ayarlar veya döndürür."error.name" propertisi tarafından 6 farklı value döndürülebilir:
*EvalError, eval() fonksiyonunda oluşan errordur.Fakat Javascript'in yeni sürümlerinde bunun yerine SyntaxError kullanılır.
*RangeError, bir "out of range" sayı oluştuğundaki errordur.
*ReferenceError, deklare edilmemiş veya illegal bir başvurunun errorudur.
*SyntaxError, bir syntax errorudur.
*TypeError, bir tür errorudur.
*URIError, encodeURI() veya decodeURI() fonksiyonlarında oluşan errordur.

*****STATEMENTS: 
*****break = Bu statement, bir switch statementından veya bir loopdan çıkar.Switch ile kullanılırsa, sadece bulunduğu switch bloğundan çıkar.Böylece mevcut blok çalışmaya devam etmez.Bir loop ile kullanılırsa, loopu keser ve çıkar; loopdan sonraki kodlar varsa da onları okumaya devam eder.Bir "label" referansı olmadan "break", sadece switch veya loop içerisinde kullanılabilir.Syntaxı:
-break;
-break labelName;
*****class = Bu statement, bir fonksiyon türüdür ancak classı başlatmak için "function" keywordu kullanmak yerine "class" keywordunu kullanırız ve içerisinde propertileri için de "constructor" metodunu çağırırız.İçerisinde bulunan constructor metodu, class objecti her initialize edildiğinde çağrılır.Fonksiyonlardan ve diğer Javascript deklarelerinden farklı olarak "class" deklerasyonları, hoisting edilmez.O yüzden classları kullanmadan önce deklare edilmeleri gerekir.Syntaxı:
-class className {
    // class body
 }
*****continue = Bu statement, belirtilen koşul oluştuğu zaman tek bir iterationı(lopp içerisindeki) keser ve devamında loopdaki bir sonraki iterationı okumaya devam eder.Sadece loopların içerisinde kullanılabilir.Labellarla kullanılabilir.Farklı looplar için farklı davranışları olabilir:
*Bir "while" loopunda önce koşul test edilir.Eğer sonuç "true" ise loop, tekrar execute edilir.
*Bir "for" loopunda önce increment statement(mesela i++,++i,..) hesaplanır.Ardından başka iterationların yapılıp yapılmayacağını belirlemek için koşul test edilir.
->Syntaxı:
-continue;
-continue labelName;
***"continue" ile "break" arasındaki fark şudur: "break", tüm loopdan çıkar; "continue" ise sadece loopun tek bir iterationından çıkar ve eğer loopun devamı varsa çalıştırmaya devam eder.
*****debugger = Bu statement, Javascriptin execute edilmesini durdurur ve eğer mevcutsa, debugging fonksiyonunu çağırır.Kod blokları içerisine bir nevi breakpointler atayabiliriz.Normalde debuggingi browserdan F12 ile konsolu seçerek aktif edebiliriz.Zaten eğer tarayıcıda debugging mevcut değilse, "debugger" statementı bir işe yaramaz.Syntaxı:
-debugger;
*****do...while = Bu statement, koşulun doğru olup olmadığını kontrol etmeden önce kod bloğunu bir kez execute eden bir loop oluşturur.Ardından da koşul doğru olduğu sürece loop, execute etmeye devam eder.Eğer koşul daima yanlışsa bile baştaki kod bloğu, en az bir kez execute edilecektir.Syntaxı:
do {
    code block to be executed
  }
  while (condition);
*****for = Bu statement, bir koşul doğru olduğu sürece execute edilen bir loop oluşturur.Koşul doğru olduğu sürece de çalışmaya devam eder; ta ki yanlış olana dek.Syntaxı:
for (statement 1; statement 2; statement 3) {
    code block to be executed
  }
*****for...in = Bu statement, bir objectin propertileri arasında dolaşır.Loopun içerisindeki ko bloğu, her property için birer kez yürütülür.Bu loopu, index sırası önemli olan arrayler için kullanmayın; bunun yerine "for" kullanın.Syntaxı:
-for (var in object) {
  code block to be executed
}
*****for...of = Bu statement, iterable bir objectin valueleri arasında dolaşır.Syntaxı:
-for (variable of iterable) {
  code block to be executed
}
*****function = Bu statement, bir fonksiyonu deklare eder.Deklare edilen fonksiyonlar, daha sonra kullanılmak üzere kaydedilir ve daha sonra çağrıldıklarında da execute edilir.Javascriptte fonksiyonlar, objecttirler ve hem propertilere hem de methodlara sahiptirler.Fonksiyonlarda value döndürebilmek için "return" kullanmalıyız.Syntaxı:
-function functionName(parameters) {
  code to be executed
}
*****if...else = Bu statement, belirtilen koşul doğru olduğu takdirde bir kod bloğunu execute eder.Koşul yanlışsa bile başka bir kod bloğunu execute edebilir.Farklı koşullara dayalı olarak farklı eylemler gerçekleştirebilen bir koşullu ifadedir.Syntaxı:
-if (condition1) {
  // block of code to be executed if condition1 is true
} else if (condition2) {
  // block of code to be executed if the condition1 is false and condition2 is true
} else {
  // block of code to be executed if the condition1 is false and condition2 is false
}
*****return = Bu statement, bir fonksiyonun execute edilmesini durdurur ve bu fonksiyondan bir value döndürebilir.Syntaxı:
-return value;
*****switch = Bu statement, farklı koşullara bağlı olarak farklı kod blokları execute eder.Birçok kod bloğundan birisini seçmek için kullanılır.Öncelikle bir statementı hesaplar.Bu statementın valuesi, daha sonra yapıdaki her bir "case"in valueleriyle karşılaştırılır.Eğer eşleşme varsa, ilgili "case"deki kod bloğunu execute eder.Genellikle "break" ve "default" keyworleriyle kullanılır:
*break, bulunduğu "switch" bloğunu execute etmeyi durdurur ve kod bloğundan çıkar.Eğer "break" eklenmezse, "switch" içerisindeki bir sonraki kod bloğu execute edilecektir.
*default, eğer herhangi bir "case" eşleşmesi olmazsa, çalıştırılacak bazı kodları belirtir."switch" içerisinde yalnızca bir adet "default" bulunabilir.
->Syntaxı:
switch(expression) {
  case n:
    code block
    break;
  case n:
    code block
    break;
  default:
    default code block
}
*****throw = Bu statement, bir error oluşturur.Bir error oluştuğunda, normalde javascript durur ve bir error mesajı oluşturur.Yani teknik olarak "Javascript, "throws" an error".Kısaca "throw" statementı, özel bir error oluşturmamıza imkan sağlar.İstisnai(özel) errorlar bir string, bir number, bir boolean veya bir object olabilir."catch" statementı ile beraber kullanılması daha doğrudur.Syntaxı:
-throw expression;
*****try...catch = Bu statement, kod bloğu çalışırken oluşabilecek errorların bir kısmını veya tamamını işler.Errorlar mesela programcı tarafından yapılan yazım hataları, yanlış kullanıcı inputu, vs olabilir."try" statementı, kodlar execute edilirken errorlara karşı test edilecek bir kod bloğu tanımlamamıza imkan sağlar."catch" statementı, "try" bloğu içerisinde bir error oluştuğunda execute edilecek bir kod bloğu oluşturmamıza imkan sağlar."finally" statementı, sonuçlardan bağımsız olarak işlemlerin en sonunda mutlak olarak execute edilecek bir kod bloğudur.Syntaxı:
-try {
  tryCode - Block of code to try
}
catch(error) {
  catchCode - Block of code to handle errors
}
finally {
  finallyCode - Block of code to be executed regardless of the try / catch result
}
*****var = Bu statement, bir değişken deklare eder.Değişkenler, bilgi depolamak için kullanılan containerlardır.Bir değişkeni oluşturduktan sonra aslında o değişken boştur yani bir valuesi yoktur.Ardından bir value ataması yaparız.Syntaxı:
-var varname = value;
*****while = Bu statement, belirtilen bir koşul doğruyken execute edilen bir loop oluşturur.Koşul doğru olduğu sürece loop, çalışmaya devam edecektir.Yalnızca koşul yanlış olduğunda duracaktır."break" veya "continue" statementlarıyla beraber kullanılabilir.Syntaxı:
-while (condition) {
  code block to be executed
}
