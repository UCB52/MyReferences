*Selectorlerde case sensitivtyi kaldırmak için "i" flagı kullanırız:
div {
    p[title="isim" i] {
        color:red;
    }    
}
*"tabindex" attribututeu, elemente focus propertisi uygular."focus" selectoruyle CSS eklenebilir.
*CSS kodlarının browserlar tarafından yorumlanırken, kullanıldıkları yerlere göre öncelik sıraları vardır:
-1. öncelik inline CSS kodlarıdır.
-2. öncelik ID tanımlı CSSler.
-3. öncelik class ile tanımlı CSSler. 
-4. öncelik selector ile tanımlı CSSler.
*"!important" ifadesini bir propertinin valuesiyle beraber kullanırsak, bu propertiye farklı yerlerde birden fazla value atanmışsa bile sadece "!important" ile kullanılan valueyi geçerli kılar.Yani öncelik sırasını en üste çıkarır.Mesela:
#deneme {
    color: red !important;  //geçerli renk
}
#deneme {
    color: blue;  //geçersiz
}
#deneme {
    color:black;  //geçersiz
}
*HTMLde her element, sayfa içerisinde dikdörtgen şeklinde alanlar işgal eder.
*Hexadecimal renk tanımlarında bazı durumlarda(çiftler halindeyse) kısayoldan yazılabilir.Mesela "#ff0088" rengi "#f08" şeklinde, "#000000" rengi "#000" şeklinde veya "#bb11aa" rengi "#b1a" şeklinde,vs..
*Width, alabildiği kadar genişlik alır.
*Height, içerik kadar yükseklik alır.
*backround-attachment propertisinde "local" ile "scroll" valueleri arasındaki fark: "scroll", sadece sayfanın çubuğuyla arkaplan içeriği scroll eder; bir elementin içerisindeyse scroll etmez."local", sayfanın veya herhangi elementin kaydırma çubuğunu kullanarak arkaplan görselini scroll edebilir. 
*border-top-right-radius, border-bottom-right-radius, border-bottom-left-radius ve border-top-left-radius propertileri 2 adet value alabilir.

*border-image belirtmeden önce herzaman normal bir "border" belirtip, bu borderın transparent yapılması gerekir.Çünkü border-image, bu transparent kısma gelecektir.
*Bir elementin içeriğini dikey ortalama yapmak için mevcut elementin line-height valuesini, bulunduğu containerın heightı kadar yapabiliriz.Böylece içeriği tüm containerı kaplayacak ve ortalanacaktır.
*Gridde repeat fonksiyonunu tekrarlar yapmak için kullanılırız.Bikaç örnek:
repeat(3, 100px); //100px 100px 100px 
repeat(2, auto 100px 250px);  // auto 100px 250px auto 100px 250px
repeat(2, 50px) repeat(3, 30px) repeat(3, 20px 10px)  // 50px 50px 30px 30px 30px 20px 10px 20px 10px 20px 10px 
*Max-content ile min-content arasındaki fark şudur: min-content, containera sığabilmek için içeriği(texti,vs) wrap edip satırlar oluşturabilir; fakat max-content, eğer mümkünse içeriği wrap etmez ve tek satırda görüntülemeye çalışır.
**Gride özel birkaç property:
*justify-items = Bu property, griddeki itemleri yatay olarak hizalar.Alabileceği valueler:
-stretch (default)
-start
-end
-center
*align-items = Bu property, griddeki itemleri dikey olarak hizalar.Alabileceği valueler:
-stretch (default)
-start
-end
-center
*place-items = Bu property, griddeki itemleri hem yatay hem dikey olarak hizalar ve bir shorthanddir.Alabileceği valueler:
-align-items
-justify-items
*justify-content = Bu property, griddeki satırları yatay olarak hizalar.Sadece itemlerin containerı tam olarak doldurmadığı durumlarda kullanılabilir.Alabileceği valueler:
-start
-end
-center
-stretch
-space-around
-space-between
-space-evenly
*align-content = Bu property, griddeki sütunları dikey olarak hizalar.Sadece itemlerin containerı tam olarak doldurmadığı durumlarda kullanılabilir.Alabileceği valueler:
-start
-end
-center
-stretch
-space-around
-space-between
-space-evenly
*place-content = Bu property, itemleri hem dikey hem de yatay olarak hizalar ve bir shorthanddir.Alabileceği valueler:
-align-content
-justify-content
*justify-self = Bu property, grid içerisindeki seçili item için hücre içerisindeki yatay hizalamayı belirtir.Alabileceği valueler:
-stretch (defualt)
-start
-end
-center
*align-self = Bu property, grid içerisindeki seçili item için hücre içerisindeki dikey hizalamayı belirtir.Alabileceği valueler:
-stretch (defualt)
-start
-end
-center
*place-self = Bu property, grid içerisindeki seçili item için hücre içerisindeki hem dikey hem yatay hizalamayı belirtir.Alabileceği valueler:
-align-self
-justify-self
*Grid yapılarında itemlerin "order" valueleri default olarak 0dır.Ordera verilen en yüksek değer en sona, en küçük değerse en başa konumlandırılır.
*Objelerin boyutları genişlik, yükseklik ve derinlikten ibarettir.
*Obje transformlarında z ekseninde dönüşümler yaparken herzaman beraberinde "perspective" kullanmaya çalışın.
*Transition kullanırken geçerli elementin propertilerine ayrı ayrı değerler verebiliriz:
.element {
    width:200px;
    height:100px;
    opacity:1;
    transition: 2s ease width, 2s ease-in height, 1s ease-in-out opacity;
}
.element:hover {
    width:300px;
    height:200px;
    opacity:0.3;
}
*initial-scale, kullanıcının ekrana yaptığı zoom değeridir.Bu zoom değerlerine göre web sayfaları, ilk yüklendiklerinde browserlarda ona göre renderlanır.Alabileceği değerler "0.1" ile "10.0" arasındadır.
*user-scalable, kullanıcının zoom yapıp yapamayacağını belirtir.
*DPİ(dots per inch), görsele dair inch başına düşen nokta sayısıdır.Yüksek dpi olan görüntüler yüksek çözünürlüklüdürler ve daha yüksek boyutludurlar.
*@media 3 farklı şekilde kullanılabilir:
1-Harici CSS dosyalarının içerisinde
2-<link> taglarının içerisinde
3-<style> taglarının içerisinde
*@media'nın örnek kullanımı: 
@media only screen and (min-width:1200px) {  //XLarge
    body {
        font-size:24px;      
    }
}
@media only screen and (min-width:992px) and (max-width:1199px) {  //Large
    body {
        font-size:22px;      
    }
}
@media only screen and (min-width:768px) and (max-width:991px) {  //Medium
    body {
        font-size:20px;      
    }
}
@media only screen and (min-width:576px) and (max-width:767px) {  //Small
    body {
        font-size:18px;      
    }
}
@media only screen and (max-width:575px) {  //XSmall
    body {
        font-size:16px;      
    }
}
